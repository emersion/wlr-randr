# wlr-randr

Utility to manage outputs of a Wayland compositor.

> **Heads up!** This project has moved over [to SourceHut](https://sr.ht/~emersion/wlr-randr/).

## Building

Install dependencies:

* meson (compile-time dependency)
* wayland

Then run:

    meson build
    ninja -C build
    build/wlr-randr
